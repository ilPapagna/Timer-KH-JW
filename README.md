# Timer-KH-JW
EN - Lightweight timer in a single HTML file
This project was created to help all Kingdom Halls and Assembly Halls that don't have high-performance computers or can't install applications.
- It was designed in a single .html file so it runs on all operating systems without any problems.
- The timer can be displayed on a second monitor, which will be visible to the speaker.
- It's possible to bring and import a timer schedule.
